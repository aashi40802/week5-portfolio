About

A bit about me

I'm Aashi, a front-end developer learning by building. I started with a single HTML file and kept going — through CSS, JavaScript, and a full multi-page restaurant website, Saffron Kitchen, with menu filtering and validated booking forms, deployed live on GitHub Pages.

I care about writing code that's readable and easy to maintain, and about designs that feel calm and work on any screen size. Right now I'm focused on modern CSS — Grid, custom properties, and animation.
